# 使用场景

## 1. 项目会议复盘

输入带说话人标签的会议逐字稿，得到摘要、共识、未解决分歧、风险点、明确 TODO 和建议。适合产品评审、项目同步、客户访谈复盘。

关键价值：把“会上说过”与“分析者建议”分开，防止建议被误记为已承诺动作。

## 2. 重要沟通的自我复盘

指定谁是“我”，查看自己的措辞之后，对方出现的可观察反应和表达变化，定位容易形成压力、模糊责任或阻塞下一步的话术。

关键价值：聚焦可调整的表达选择，不猜测对方的真实内心或隐藏意图。

## 3. 访谈与调研分析

按参与者拆解事实主张、观点、态度与保留条件，观察问题设计之后的澄清、抵触、对齐或转题。

关键价值：保留混合表达与不确定性，减少把偏好当事实或把沉默当同意的风险。

## 4. 单人语音或独白

对单人文本或获授权的录音做结构化复盘，发现情绪表达转折、自我修正、否定、反问、模糊表达、责任表达和行动导向。

关键价值：单人输入自动将唯一说话人视为“我”；文本模式只做语言侧 VAD，不臆测声音特征。

## 5. 沟通陪练

从真实对话中选择一个表达卡点，保留原场景让用户重新回答，再按清晰度、承接、情绪负荷、证据分离和行动性反馈。

关键价值：一轮一个问题，先练再评，不提前生成用户答案。

## 6. 长期团队关系观察（需显式授权）

多个单次报告可在用户明确同意后形成带来源、有效期、置信度和删除关联的长期观察。单次分析默认不写入关系本体或习惯偏好。

关键价值：长期信息可追溯、可撤销；不得仅凭声纹跨会话绑定身份。

## 不适用场景

- 心理疾病诊断、人格定型或测谎；
- 未经同意的录音、上传或长期画像；
- 根据情绪推断自动做招聘、绩效、保险、教育、医疗或法律决定；
- 只有文本却要求音高、能量、语速、停顿或音质结论；
- 用自我回声设计操纵、施压或欺骗话术。
