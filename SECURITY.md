# 安全与隐私政策

请不要在公开 Issue、Pull Request、示例或测试夹具中上传真实的私人录音、逐字稿、身份信息、访问令牌或分析结果。

## 私下报告

在 GitHub 仓库的 **Security** 页面使用 “Report a vulnerability” 私下提交问题。报告应包括受影响版本、复现条件、潜在影响和建议修复；不要附带不必要的真实个人数据。

## 范围

我们特别关注：

- 原始音视频或逐字稿意外泄露；
- 未经同意写入长期记忆或外部系统；
- 多说话人身份误绑定；
- 文本模式伪造声学结论；
- 合同校验绕过造成的错误动作状态或证据引用；
- 将表达分析误用于医疗、就业、教育、保险、法律或安全决策。

本仓库当前不托管分析服务，也不收集遥测。使用者仍需确保录音、存储和处理符合所在地区法律及参与者授权。
