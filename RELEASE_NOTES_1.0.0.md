# memEcho Skills 1.0.0

This is the first stable community release of `memecho-analyze-conversation`.

## Included capabilities

- Meeting summaries with consensus, disagreements, confirmed actions, and proposed recommendations.
- Per-speaker fact claims, opinions, attitudes, and observable interaction influence.
- Evidence-linked VAD trajectories with strict text-only and acoustic-signal boundaries.
- Self-echo analysis from a user-confirmed viewpoint.
- Interactive coaching that waits for a real response before scoring.
- Portable JSON contracts, validator, examples, and request-construction CLI.

## Distribution

Use the attached `memecho-analyze-conversation-v1.0.0.zip` for SkillHub and other Agent Skills-compatible platforms. `SKILL.md` is at the archive root. Verify the download against `SHA256SUMS.txt` before installation.

Full audio analysis is not bundled in this release. It requires an authorized memEcho service or a compatible local media adapter. Text-only analysis never invents acoustic features.
